Bienvenido al Taller de Consumo de API

Este proyecto interactúa con la API pública JSONPlaceholder y realiza operaciones CRUD sobre 4 entidades.

Posts

Publicaciones creadas por usuarios. Puedes crear, editar, ver y eliminar entradas.

Comments

Comentarios relacionados con los posts. Muestra los mensajes de los usuarios.

Albums

Álbumes que agrupan fotos por usuario. CRUD completo disponible.

Photos

Imágenes asociadas a los álbumes. Visualiza thumbnails.

Proyecto académico - Consumo de API REST con Angular y JSONPlaceholder
